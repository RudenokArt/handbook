Посмотреть залогиненных пользователей
who

Посмотреть текущего пользователя
whoami

Под кем вошел текущий пользователь:
logname

Просмотр всех пользователей
vim etc/passwd

Просмотр хэша паролей пользователей
vim etc/shadow

Создать пользователя test
useradd test

Установить пароль пользователю test (или другие данные пользователя - по ключам)
passwd test

Просмотр данных пользователя (можно менять этой же командой с ключами)
chage -l test

Смена прав на файл
chmod 777 test

Смена владельца и группы файла
chown bitrix:bitrix test
