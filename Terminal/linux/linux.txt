текущий путь
pwd

Текущий каталог (состав)
ls

Текущий каталог (состав) - вертикально
ls -l

Текущий каталог (состав) - со скрытыми файлами
ls -a

Текущий каталог рекурсивно с подпапками
ls -laR

Просмотр подкаталога
ls /home

Копирование файла (что копируем (файл 500.html), куда копируем (в папку bitrix))
cp 500.html bitrix

Копирование директории (папку test в папку bitrix)
cp -R test bitrix

Удаление файла
rm test.txt

Удаление директории
rm -R test

Информация по файлу
file error.log

Создание файла
touch test.txt

Переименовать файл
mv test.txt text.txt

Создать папку
mkdir test

Создать папку с подкаталогами
mkdir test1/test2

заити в редактор:
vi
vim
включить insetr mode:
i
сохранить
Esc + :w
сохранить с переименованием:
Esc + :w /путь/к_файлу.txt
сохранить и выйти:
Esc + :wq
выйти:
Esc + :q
